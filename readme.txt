To use the bot you will need Session.json
to get that scan the qr code generated by https://replit.com/@vatsdavesh/Zero-Chatbot-Qr-Code-Generator?v=1
then you will get the session.jsson on your whatsapp add this to your forked repo or in your pc
then run npm install
then npm start if you dont want it to stay running
if you want it to let it stay running forever
use forever start index.js
if you want to host this on heroku see the readme.md
